# Code of Conduct

## Our pledge

We want participation in this project to be respectful, constructive, and welcoming for everyone.

## Expected behavior

- Discuss ideas and technical trade-offs in good faith.
- Give feedback on code and behavior, not personal characteristics.
- Respect privacy. Do not post personal data, captured images, access credentials, or confidential system details.
- Assume good intent and ask for clarification before escalating disagreements.

## Unacceptable behavior

Harassment, discrimination, threats, personal attacks, deliberate exposure of private data, and repeated disruption of project discussion are not acceptable.

## Reporting

Report conduct concerns privately to the repository maintainers. Security and privacy vulnerabilities should follow [SECURITY.md](SECURITY.md).
